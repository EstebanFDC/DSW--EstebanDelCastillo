Propósito del Sitio:

El propósito del sitio es tener un acercamiento a las herramientas que se utilizan para diseñar sitios web utilizando lenguaje HTML, como son los Frameworks CSS y cuyo uso adecuado de estos permite generar una ruta adecuada hacia un óptimo Posicionamiento Web.

FRAMEWORKS CSS

En sus inicios los sitios web estaban conformados por información que permanecía estática, pero que con el paso del tiempo se fueron transformando, volviéndose más dinámicas y dando lugar a aplicaciones web, las cuales permitían realizar las tareas de la misma manera que se realizaban desde el escritorio de un computador. Dichos avances gestaron el espacio propicio para generar nuevas maneras de optimizar el trabajo del desarrollo y la programación, es aquí donde nacen los Frameworks CSS.
Podría entonces definirse los Frameworks CSS como una estructura de software compuesta por componentes personalizables e intercambiables, utilizados en la creación de aplicaciones o sitios web, cuyo objetivo es acelerar el proceso de desarrollo, reutilizando códigos ya existentes, así como promover prácticas para generar patrones de uso.

POSICIONAMIENTO WEB

Cuando hablamos de posicionar estamos hablando de colocar alguna cosa en un lugar óptimo. Dentro de lo que respecta a los sitios web y en el ámbito de la world wide web, posicionar un sitio significa optimizarlo para que aparezca en las primeras posiciones de las páginas de resultados de los motores de búsqueda.
Entonces podríamos definir al posicionamiento web como el conjunto de técnicas y procedimientos que tienen como objetivo y finalidad dar a u sitio web la máxima visibilidad posible dentro de la internet.
