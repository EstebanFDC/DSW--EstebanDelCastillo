Propósito del Sitio:

El propósito del sitio es tener un acercamiento a las herramientas que se utilizan para diseñar sitios web utilizando lenguaje HTML, como son los Frameworks CSS y cuyo uso adecuado de estos permite generar una ruta adecuada hacia un óptimo Posicionamiento Web.

FRAMEWORKS CSS



En un inicio los sitios web estaban construidos y conformados por información que se mostraba de manera estática para el usuario. Dicha necesidad de encontrar maneras de dinamizar este proceso informativo dio lugar a que aparecieran diversas aplicaciones web, las cuales servirían de herramientas para la elaboración y diseño de sitios web, cuya principal función sería crear una verdadera experiencia de usuario. Todos estos avances gestaron el espacio propicio para generar nuevas maneras de optimizar el trabajo del desarrollo y la programación, es aquí donde nacen los Frameworks CSS.
Podríamos definir los Frameworks CSS, como una biblioteca de estilos que crea una estructura de software, sobre la cual podemos implementar un diseño para un sitio web y nos sirve de base en general para el desarrollo de softwares, destinados a suplir las necesidades de los usuarios.

La mayoría de Frameworks CSS, proporcionan diversas utilidades para cada uno de los aspectos que se tienen en cuenta para el desarrollo de un diseño web; pero también existen unos cuantos con un propósito enfocado en atender necesidades específicas del programador por ejemplo: Frameworks orientados al control de la parte gráfica de la interfaz del usuario, a la tipografía, al desarrollo de rejillas, etc.

Aunque existen diversas opciones de Frameworks CSS, se destacan entre los más utilizados: Awsm.css, Bootstrap, Chopstick, Concise, Flexify, Ink, Kube, Boxysheets, Baseguide.

Ventajas

Algunas de las ventajas que se obtiene mediante el uso de frameworks CSS son:

• Permite implementar diseños web de manera rápida y cómoda.
• Ayudan a que el diseño implementado pueda funcionar en variedad de navegadores.
• Cumplen con normas estándar lo que garantiza cierta seguridad.
• Esta probado anticipadamente para asegurarnos de que no se presenten errores a la hora de aplicarlos.

Desventajas

Aunque son mínimas, hay que tener en cuenta ciertas consideraciones que pueden convertirse en desventajas si no se tiene cuidado al hacer uso de los Frameworks CSS, tales como:

• El uso excesivo de códigos del Framework CSS que no se necesite dentro de nuestra propuesta de diseño web, consume más recursos y por ende el tiempo de carga aumenta.
• Al tener una plantilla que está prediseñada, se puede limitar al programador en cuanto a posibilidades de elección para el diseño, es decir solamente se basaría en lo que está ya hecho y no lo que se puede llegar hacer.
• Pueden presentarse algunos defectos en la visualización del sitio web y para quien está diseñando es difícil saber donde se encuentra el fallo.


POSICIONAMIENTO WEB



Cuando hablamos de posicionar estamos hablando de colocar alguna cosa en un lugar óptimo. Dentro de lo que respecta a la web podríamos decir que el posicionamiento web son el conjunto de técnicas y procedimientos que tienen como objetivo y finalidad optimizar un sitio web para poder generar una mayor visibilidad dentro de los buscadores dentro de internet, en otras palabras lo que se pretende es optimizar la experiencia de búsqueda por parte del usuario.

Dentro de esas estrategias existen elementos que ayudan a impulsar un buen posicionamiento web, entre ellas se destacan:

Las palabras claves o Keywords:
Permiten generar una coincidencia con los términos que utilizan los usuarios cuando hacen una búsqueda, por lo tanto entre mayor palabras claves más posibilidades de que accedan al sitio web que estamos desarrollando.

El contenido:
Entre más original sea el contenido, mayores son las posibilidades de lograr un óptimos posicionamiento web, debido a que los buscadores hacen comparaciones e incluso pueden llegar a penalizar cuando un contenido es copia de otro.

Actualización de publicaciones:
Esto va de la mano con el contenido, es decir entre menor sea el tiempo en que se muestre más contenido, mucho mayor será el posicionamiento que le puedan dar al sitio web.

Así también existen algunas malas practicas que frenan la posibilidad de generar un óptimo posicionamiento web, entre ellas tenemos:

Exceso de publicidad:
Esto va dirigido específicamente a estropear la experiencia de usuario, debido a que muchas veces se generan banners o propagandas alusivas a publicidad en gran cantidad y que la mayoría de las veces no tiene nada que ver con el contenido que se esta ofreciendo.

Contenido duplicado:
Como se hablaba anteriormente, copiar contenido de otro sitio para mostrarlo en uno propio, además de poder acarrear sanciones legales, crea la mala imagen ante el usuario de que solamente se esta copiando y no generando información.

Generar enlaces sin contenido
Hacer un uso excesivo de keywords, y colocar enlaces sin contenido solamente para generar posicionamiento, se convierte en SPAM, haciendo que los buscadores dejen de ofrecer el sitio web como alternativa a una búsqueda realizada por un usuario.
